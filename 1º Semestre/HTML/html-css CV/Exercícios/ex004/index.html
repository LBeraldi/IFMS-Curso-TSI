Neste exercício, vamos entender um pocuo como funciona as tags de destaque de texto

Negrito / destaque

Nesta frase temos um temo em destaque utilizando a tag B (Não semântica)

Nesta frase temos um termo em destaque utilizando a tag STRONG (semântica)

Itálico / Ênfase

Nesta frase temo um texto em itálico utilizando a tag I (Não semântica)

Nesta frase temos um texto em itálico utilizando a tag EM (semântica)

Texto marcado

Podemos também marcar um texto através da tag MARK

Podemos utilizar algumas tags em css para mudar a cor da marcação, desta forma temos um texto marcado utilizando a cor de sua preferência

Texto maior e menor

Podemos destacar um texto ou deixa-lo menor utilizando as tags BIG e SMALL

Texto deletado

Podemos criar um texto deletado, que pode ser lido, porém não considerado com a tag DEL

Texto inserido

Podemos também criar um texto inserido utilizando a tag INS

Texto sobrescrito

Podemos utiliza um texto sobrescrito por exemplo: x20-3

Texto subescrito

Também podemos criar um texto subescrito por exemplo: H2O
